# 13TANG LIVESHOW

滚动动画酒水单项目，演示一个适合酒吧/Livehouse 的移动优先单页菜单体验。

## 功能亮点

- 固定头部与可滚动分类导航
- 页面滚动进度条与回到顶部按钮
- 可视化卡片动画与多种 entrance 效果
- 视差粒子背景与 `HTML5` 语义结构
- 响应式布局，兼顾移动端体验

## 本地调试

1. 直接打开 `index.html`
2. 或使用本地静态服务器运行

```powershell
cd /d d:\13T
your-static-server-command
```
